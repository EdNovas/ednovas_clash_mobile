# 隐私政策 / Privacy Policy

**最后更新：2024年12月**

## 中文版

### 1. 信息收集

EdNovas云 仅收集以下信息：
- **账号信息**：用于验证您的 EdNovas 订阅状态
- **网络流量统计**：仅用于显示使用量，数据存储在本地设备

### 2. 信息使用

收集的信息仅用于：
- 验证订阅有效性
- 提供 VPN 代理服务
- 显示使用统计

### 3. 数据存储

- 账号凭据加密存储在本地设备
- 不存储任何网络流量内容
- 不记录访问的网站或 IP 地址

### 4. 第三方服务

本应用连接 EdNovas 服务器获取配置，不使用其他第三方分析或广告服务。

### 5. 用户权利

您可以随时：
- 退出账号清除本地数据
- 卸载应用删除所有相关信息

### 6. 联系方式

如有隐私相关问题，请联系：support@ednovas.com

## English Version

### 1. Information Collection

EdNovas Cloud only collects:
- **Account information**: To verify your EdNovas subscription status
- **Network traffic statistics**: Only for displaying usage, stored locally on device

### 2. Use of Information

Collected information is only used for:
- Subscription validation
- Providing VPN proxy service
- Displaying usage statistics

### 3. Data Storage

- Account credentials are encrypted and stored locally
- No network traffic content is stored
- No visited websites or IP addresses are logged

### 4. Third-Party Services

This app connects to EdNovas servers for configuration only. No third-party analytics or advertising services are used.

### 5. User Rights

You can at any time:
- Log out to clear local data
- Uninstall the app to delete all related information

### 6. Contact

For privacy concerns, please contact: support@ednovas.com
